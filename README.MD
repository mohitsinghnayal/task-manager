# TASK MANAGER APP

# HOW TO RUN PROJECT

1- install node_modules using npm i in the root folder.
2 - run npm start or npm run dev in the command terminal to run the project

# APIS

There are two types of modules:

1- User auth module
    - here you can register, login and logout a user.

2- Tasks module
    - here you can create, get all tasks and update a task


# TEST CASES
Testing framework used is JEST

1-  user module test cases
    - to run this test run "npm test ./tests/user.test.js"

2- task module test cases
    - to run this test run "npm test ./tests/task.test.js"
